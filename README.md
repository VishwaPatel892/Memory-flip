
# 🧠 Memory_Flip_Game  

A fun and engaging **Memory Flip Game** designed to test and improve your memory skills.  
Flip the cards, find matching pairs, and complete the game in the fewest moves possible!

---

## 🚀 Features  

- 🃏 Card flipping animation  
- 🧠 Enhances memory and concentration  
- 🎯 Match pairs of identical cards  
- ⏱️ Tracks moves (optional timer support)  
- 🔄 Restart / New Game option  
- 🎨 Simple and responsive UI  

---

## 🛠️ Tech Stack  

- **HTML**  
- **CSS**  
- **JavaScript**  

*(Can be upgraded using React or other frameworks)*  

---

## 🎮 How to Play  

1. The game starts with all cards placed face down.  
2. Click on any card to flip it.  
3. Flip another card to find its match.  
4.  
   - ✅ If both cards match → They stay flipped  
   - ❌ If not → They flip back  
5. Continue until all pairs are matched.  
6. Click **Restart / New Game** to play again.
